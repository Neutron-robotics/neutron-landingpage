- **Easy to learn:** The interface has been built with a particular focus on simplicity. Things don't need to look complicated when they can be simplified, allowing users from different background to collaborate without being lost with details.
- **Beautiful by default:** Profit of a modern and purified interface that has been built implementing modern web design standards. You will love using the platform.
- **Customizability** Your system is unique, and so your interface should be. Once you've setup the software for your usage, the experience will always be the same.
- **Portability** You can securly access your system from any devices from anywhere thanks to the use of web technologies.
